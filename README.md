4. 최소 공통 조상(LCA : Lowest Common Ancestor)

8. 플루이드-워셜 알고리즘(Floyd-Warshall Algorithm)

1. 삽입 정렬(Insertion Sort)

2. 퀵 정렬(Quick Sort)

3. 선택 정렬(Selection Sort)

4. 힙 정렬(Heap Sort)

5. 계수 정렬(Counting Sort)

6. 합병 정렬(Merge Sort)
